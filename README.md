# Sistema de registro de usuarios

Proyecto académico desarrollado como entrega universitaria para la asignatura impartida por el **profesor Mateo**.

## Descripción

Aplicación web orientada a la **captura, validación y gestión básica de datos de usuarios** (registro). El objetivo es demostrar el manejo de formularios, la separación de estructura (HTML), presentación (CSS) y comportamiento (JavaScript), y buenas prácticas en la experiencia de usuario al completar un alta.

## Alcance previsto

- Formulario de registro con campos definidos según los requisitos del trabajo.
- Validación en el cliente (y, si aplica, criterios acordados con el docente).
- Interfaz clara y usable, coherente con los lineamientos de la asignatura.

## Cómo ejecutar el proyecto

1. Clonar o descargar este repositorio.
2. Abrir el archivo principal HTML en un navegador moderno (por ejemplo, doble clic en `base html.html` o usar la configuración de depuración en VS Code / Cursor si está disponible).

> Si el profesor indicó otro método de entrega o evaluación (por ejemplo, despliegue estático o comprimido), seguir esas instrucciones.
